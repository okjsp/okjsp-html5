Multiflex-4 Overview

Flexible and robust template

Now substantially improved and simplified

Multiflex-4 is the fourth generation Multiflex template, with the header code being the major change since Multiflex-3. The new header is more in line with current design trends. The core code is almost identical to Multiflex-3.10, with the major difference being in the CSS code for the header. Outsie the header, both the CSS and HTML codes have remained virtally unchanged. Multiflex-4 does not offer multiple headers, but only one standard header. Check the navigation menu to discover all about the new Multiflex-4!

Extensive browser support

Tested for all common browsers and systems

Multiflex-4 has been tested live for all major browsers, and functions properly for all currently used browsers. Only exception is the very outdated legacy browser IE5.x on Windows and Mac, which is used by about 1.5% of all websurfers globally.

See links in navigation bar for more details.

Code Structure

The CSS code has been modularized into two files for easier maintenance:

1) setup.css defines the basic layout structure of the page.

2) text.css defines the text formats available for the page main section.

About me

My name is Gerhard, and you may be surprised to hear that for my real job I work as a full-time pastor for a Christian church. Feel free to visit our global church website.

In 2006 I decided to start my own website. When surfing around I didn't find the template I needed. So I began to develop my own, with trials, errors and many long nights. About half a dozen experiments and six months later, I feel that I am coming closer to the design I need! But I will try to push it one step further, and make a liquid layout.

I have profited much from others who have shared their CSS-knowledge on the internet. The best way to show my appreciation is by sharing my results with others. So if this template is what you were looking for, then feel free to use and enjoy it!
